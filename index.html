02 · Feedback-Driven Design
02 / 05

销售 Leader 反馈 Driven Design

本项目每一个关键设计决策都能追溯到一线销售 Leader 的真实反馈。以下三条反馈是 v2.0 核心升级的源头—— 工具要服务于销售的真实工作方式，不是反过来让销售适应工具。

Feedback · 01
"客户不会问某只 ETF 适不适合营销，客户提出来的都适合。客户会问——最近你们哪些赛道比较热？"
→ DESIGN RESPONSE
新增模式四 · 赛道推荐，支持"今天哪些赛道最热"的反向查询。基于 follow7d 行为数据反向映射赛道 + 基金公司清单。
Feedback · 02
"中间最有价值的就是雪球的数据分析部分。"
→ DESIGN RESPONSE
引擎 B 升级为三模块（讨论 50% + 行为 40% + KOL 10%），新增 follow7d / hot_stock 等基金公司后台看不到的独有信号。
Feedback · 03
"国泰富国这种是投研决策，会直接给想投的产品，我们给意见怎么投。其他客户会参考我们的赛道建议。"
→ DESIGN RESPONSE
客户三分层（投研驱动 / 渠道驱动 / 新进小厂），态势 × 客户 = 9 套话术变体，同一份数据不同对接人用不同话术。
03 · Four Operating Modes
03 / 05

四种使用模式 — Usage Modes

模式覆盖销售 四类高频场景——跟单只 ETF 客户会、扫一遍基金公司全产品线、判断赛道竞争格局、做日报看今日热度。 每种模式自动识别、自动路由，销售用自然语言就能触发。

— Mode I

单品诊断

Single ETF Analysis

一只 ETF 的完整营销诊断，三引擎全分析 + 客户话术 + SKU 组合，一份能直接带去客户会的机会简报。

"159995 适合持营吗" "给我华夏芯片 ETF 的客户弹药"
CORE OUTPUT
  • 三引擎得分 + 共振判定 + 竞争态势
  • 客户类型适配话术（护城河型独立分支）
  • 3 个核心 SKU 推荐 + 真实案例背书
— Mode II

全司地图

Fund House Portfolio Scan

某家基金公司旗下全部 ETF 扫描，自动按赛道聚类 + 分层排优先级 + 跨产品协同建议。

"华夏基金旗下 ETF 扫一遍" "易方达哪些 ETF 值得重点推"
CORE OUTPUT
  • ETF 优先级矩阵 · 四档分层
  • 护城河型产品独立标注
  • 共享 KOL 池 + 联合话题机会
— Mode III

竞品格局

Peer Competitive Landscape

同一指数赛道的横向竞争格局 + 白地分析——识别"社区已讨论、竞品未跟上"的窗口。

"中证 A500 ETF 赛道比一比" "AI 算力 ETF 谁在领跑"
CORE OUTPUT
  • 竞争格局矩阵（全家横向对比）
  • 白地 ETF 清单 + 饥渴度指数
  • 领先者 / 红海 / 落后者分别话术
v2.0 核心新增
— Mode IV

赛道推荐

Sector Momentum · Reverse Query

销售最高频的"今天哪些赛道热"的反向查询——基于站内行为数据反向映射赛道和对应基金公司客户。

"今天站内哪些 ETF 赛道最热" "给我今日 ETF 营销日报"
CORE OUTPUT
  • 市场背景 + TOP 5 热赛道 + 热点催化
  • 白地赛道清单 · 冷门赛道清单
  • 今日建议优先触达客户（反查映射）
04 · Scoring Framework
04 / 05

三引擎评分体系 Engines

综合分 = 0.35×A + 0.40×B + 0.25×C。引擎 B 权重最高，因为这是雪球独有价值—— 基金公司在自家后台看不到的上游生态数据，正是我们的差异化弹药。

ENGINE A35%

客观行情

Market Signals

时效性信号，基金公司自己也有。但我们 扫全市场更快——同样的涨跌幅 / 成交额 / 换手率 / 溢折价 / 振幅 / 相对强度，我们跑 60 日 K 线全市场横向扫描。

—涨跌幅（日 / 5日 / 20日）
—成交额放大倍数 · 换手率
—溢折价 · 振幅 · 相对强度
v2.1 升级
ENGINE B40%

生态势能

Ecosystem Energy

雪球独有的差异化弹药。v2.0 新增行为势能模块，让 follow7d（7 日新增关注）这种聪明资金注意力信号成为最前置的持营判断依据。

50%B1 · 讨论势能（L1/L2/L3/L4 四层聚合）
40%B2 · 行为势能（follow7d + hot_stock）
10%B3 · KOL 卷入度（跨层扩散）
ENGINE C25%

竞品对标

Competitive Landscape

同指数 ETF 横向比较——触发基金公司"看对手动作做决策"的心理，也是白地识别和态势分类（领先者 / 红海 / 落后者）的基础。

40分规模对标（同指数排名 + 领先倍数）
30分费率对比（管理费 + 托管费）
30分近期表现超额（相对中位数）
